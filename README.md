# Money Ledger – Smart Expense Tracker App

Money Ledger is a Flutter-based mobile application designed to help users **track, categorize, and analyze their daily expenses**. The app provides a clear overview of spending habits through **real-time data visualization**, enabling better financial decision-making.

---

## Features

### Expense Tracking
- Add and manage daily expenses بسهولة  
- Categorize spending into:
  - Groceries  
  - Travel  
  - Food  
  - Utilities  
  - Custom categories  

---

### Real-Time Analytics
- Visual representation of expenses using **interactive charts/graphs**  
- Track spending distribution across different categories  
- Helps users understand financial patterns instantly  

---

### Category-Based Management
- Structured expense logging  
- Easy filtering and organization of transactions  
- Clean and intuitive UI for better usability  

---

### User-Friendly Interface
- Simple and minimal design  
- Smooth navigation and responsive layout  
- Focused on **ease of use and clarity**
